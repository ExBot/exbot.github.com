Aquarium Scheme

方案一、机器虫展区

整体描述

  1. 机器虫展区为仿自然环境的玻璃罩隔离区域
  2. 5大类共70个小型昆虫造型机器人在展区内自由爬行
  3. 3大类共12台六足机器人与游客互动

小型昆虫造型机器人

70个小型昆虫机器人分为5类。

  1. 4个声控光感电子蟹：当有击掌声或其他的声响，它会停止前进，并开始绕足，悄悄改变行进的方向。爬行到光线很暗的环境中，会停止前进，直到听到击掌声或是光线变亮才会继续爬行。

  2. 21个声控触感电子昆虫：前面一对触须采用弹簧线制作，触须受力时产生信号，控制其后退转弯，因此，遇到障碍物它会自动躲避。内置声控传感器，当你发出声响比如拍手，它会改变方向跑掉。

  3. 15个红外感应电子毛毛虫：蠕动前进，头部红外头感应到障碍物能转向。

    视频链接

  4. 12个触须感应电子蚂蚁：它能能透过触觉感应器感知障碍物，当触角碰到障碍物时就会自动倒退然后绕开，或者改变方向逃跑。

  5. 18个电子蟑螂：四处乱撞，撞到墙就转向。

    视频链接

主材料估价：664 + 4821 + 1518 + 1218 + 4.5*18 = 1839元

六足机器人

12台六足机器人分为3类。

  1. 4台表演型机器人：电脑控制其在展区内活动。

    视频链接

  2. 4台手柄遥控六足机器人（配置2自由度机械手）：游客可以控制六足机器人移动，做出特定动作，并控制机械手抓取其他机器人；管理员（游客）可以用它来抓取不动（电池耗光）了的小型昆虫机器人，并把抓取到的昆虫机器人丢入回收口（方便管理员给其换电池），增强娱乐性。

    淘宝及视频链接

  3. 4台Kinect互动+语音互动六足机器人：当游客（对着Kinect）向它招手时，它能走过来；当游客向它做动作时，它能做出反应（伸爪，站立等）；当游客用语音喊话时，它能做出反应。

组件：

  1. 主机体1。560元：全身结构+19个GS9018舵机，不带控制电路。舵机扭矩：1.8kg-cm@6V，1.6kg-cm@4.8V。
  2. 主机体2。1520元 = 350 + 18*65：全身结构 + 18个TR213舵机，不带控制电路。舵机扭矩13kg·cm，性能优于MG995。
  3. 主机体3。1014元 = 330 + 18*38：全身结构 + 18个 MG995 舵机，不带控制电路。舵机扭矩13kg·cm。

  4. 机械臂。239元：结构件 + 2个TR205舵机。舵机扭矩：6.5kg·cm。

  5. 无线手柄套件。270元。

  6. 32路舵机控制板。168元。
  7. 7.4V 5200MAH 30C 聚合物2s锂电池。120元。
  8. 平衡充电器。160元。
  9. Kinect。760元。
  10. WIFI模块/蓝牙模块。待定
  11. Arduino主控板，上位机。待定

主材料估价：5604 + 15204 + 10144 + 2394 + 2704 + 16812 + 12016 + 1604 + 7604 + 50012 = 28028元

备注

主材料机器人部分估价为：1839 + 28028 = 29867元

除此以外，还需要部分加工设备，展台制作，电脑等。

  3. 超声波避障模块。28元。淘宝链接
  4. 无线手柄套件。270元。
  5. 5V 3Ah锂聚合物电池。46元。淘宝链接
  6. Arduino控制板。待定。

主材料估价： 2556 + 795 + 2836 + 2706 + 466 + 300*6 = 7029元
